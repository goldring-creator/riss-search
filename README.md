# RISS 논문 수집기

RISS(학술연구정보서비스)에서 키워드로 국내학술지 논문을 자동 수집하고 PDF를 다운로드하는 데스크탑 앱입니다.

> ⚠️ **학위논문은 수집되지 않습니다.** 국내 학술지 논문만 대상입니다.  
> ⚠️ **macOS 전용**입니다.

---

## 설치 방법

### 1. Node.js 설치

터미널에서 확인:
```bash
node -v
```
버전이 표시되면 OK. 없으면 [nodejs.org](https://nodejs.org) 에서 **LTS** 버전 설치.

---

### 2. 다운로드

이 페이지 상단 **Code → Download ZIP** → 압축 해제

---

### 3. 보안 설정 해제 ← 필수

GitHub에서 내려받은 파일은 macOS가 실행을 차단합니다.  
**압축 해제한 폴더 안**에서 터미널을 열고 아래 명령어를 실행하세요:

```bash
xattr -dr com.apple.quarantine .
```

> 터미널에서 폴더 열기: Finder에서 폴더 우클릭 → 폴더에서 새 터미널 열기

---

### 4. 실행

`RISS 논문수집기.app`을 **우클릭 → 열기** 선택.  
(첫 실행 시에만 우클릭 필요. 이후에는 더블클릭으로 바로 실행됩니다.)

- 브라우저가 자동으로 열리며 앱 화면이 표시됩니다
- 최초 실행 시 패키지·브라우저 설치가 자동 진행됩니다 (1~3분)

---

## 사용 방법

### ① 도서관 ID/PW 저장

소속 도서관 이용자 ID와 PW를 입력하고 **저장** 클릭.  
처음 저장 시 Keychain 팝업 → **"항상 허용"** 클릭.

### ② 키워드 입력

Enter 또는 쉼표로 키워드 태그 추가. 복수 키워드는 각각 검색 후 병합됩니다.

### ③ 필터 설정 (선택)

| 옵션 | 설명 |
|------|------|
| 수집 페이지 수 | 1페이지 ≈ 10건 |
| 연도 범위 | 빈칸 = 전체 |
| KCI 등재만 | KCI 등재 논문으로 한정 |
| 정렬 방식 | 정확도 / 최신 / 인기도 |
| 상위 N건 | 피인용수 기준 상위 N건만 |
| 제외 키워드 | 제목/초록 포함 시 제외 |

### ④ 저장 위치 → 수집 시작

폴더 선택 후 **수집 시작** 버튼 클릭. 실시간 로그로 진행 상황 확인.

---

## Claude API 분류 기능 (선택)

1. [console.anthropic.com](https://console.anthropic.com) 에서 API 키 발급
2. 앱의 Claude API 키 란에 입력 후 저장
3. 필터 패널에서 **"분류 건너뜀"** 체크 해제

---

## 출력 파일

```
저장위치/
├── metadata.json
├── report.csv         (분류 시)
└── pdfs/
    └── 저자명(연도). 논문제목.pdf
```

## 자주 묻는 질문

**Q. 더블클릭해도 열리지 않습니다.**  
A. 우클릭 → 열기를 시도하세요. 그래도 안 되면 설치 3단계(xattr 명령어)를 실행했는지 확인하세요.

**Q. 도서관 로그인이 실패합니다.**  
A. RISS 직접 계정이 아닌 소속 기관 도서관 계정을 입력하세요.

**Q. Keychain 팝업이 매번 뜹니다.**  
A. "항상 허용"을 클릭하면 이후 팝업이 뜨지 않습니다.

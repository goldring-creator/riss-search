Claude CLI 설치

Claude CLI(Claude Code)를 설치하면 API 키 없이도 논문 자동 분류 기능을 사용할 수 있습니다.
Claude.ai 계정만 있으면 무료로 사용 가능합니다.

CLI vs API 키 — 어떤 걸 써야 하나요?
Claude CLI는 무료로 사용할 수 있지만 속도가 느릴 수 있고, API 키는 유료이지만 빠르고 안정적입니다. 처음 사용하신다면 CLI를 추천합니다.
Step 01

Node.js 설치 확인

Claude CLI는 Node.js 18 이상이 필요합니다. 터미널에서 버전을 확인하세요.

node --version

버전이 표시되지 않거나 18 미만이면 nodejs.org에서 LTS 버전을 설치하세요.

Step 02

Claude CLI 설치

npm으로 전역 설치합니다. 터미널에 아래 명령어를 입력하세요.

npm install -g @anthropic-ai/claude-code
macOS/Linux에서 권한 오류가 나면 앞에 sudo를 붙이거나, nvm으로 Node.js를 설치하면 sudo 없이 사용할 수 있습니다.
Step 03

로그인

설치 후 아래 명령어로 Claude.ai 계정에 로그인합니다. 브라우저가 열리면 계정으로 인증하세요.

claude login

로그인이 완료되면 터미널에 Logged in 메시지가 표시됩니다.

Step 04

설치 확인

정상 설치 여부를 확인합니다.

claude --version

버전 번호가 출력되면 완료입니다. 앱으로 돌아가 분류 건너뜀 체크를 해제하면 자동 분류가 활성화됩니다.

API 키 발급

Anthropic Console에서 API 키를 발급받으면 CLI 없이도 빠르게 논문 자동 분류를 사용할 수 있습니다.
유료 서비스이며, 사용량에 따라 과금됩니다.

요금 안내
논문 100편 분류 기준 약 $0.05~$0.20 수준입니다 (모델·길이에 따라 다름). Anthropic은 신규 가입 시 무료 크레딧을 제공하므로 처음에는 무료로 테스트할 수 있습니다.
Step 01

Anthropic Console 접속

아래 링크에서 Anthropic Console에 접속합니다. 아직 계정이 없으면 이메일로 가입하세요.

console.anthropic.com →

Step 02

API Keys 메뉴 이동

로그인 후 왼쪽 사이드바에서 API Keys를 클릭합니다.

처음 사용 시 결제 수단 등록을 요구할 수 있습니다. 무료 크레딧 범위 내에서는 실제 과금이 발생하지 않습니다.
Step 03

새 키 생성

Create Key 버튼을 클릭하고 이름을 입력합니다 (예: riss-search).

생성된 키는 sk-ant-로 시작합니다. 이 화면에서만 전체 키를 볼 수 있으니 바로 복사해두세요.

주의: API 키는 비밀번호처럼 관리하세요. 이 앱에 저장하면 macOS Keychain에 암호화되어 보관됩니다.
Step 04

앱에 키 입력

앱으로 돌아가 검색 필터 및 옵션 → Claude API 키 입력란에 붙여넣고 저장을 클릭합니다.

분류 건너뜀 체크를 해제하면 수집 후 자동 분류가 실행됩니다.

검색 로직

키워드·필터 설정이 실제로 어떻게 동작하는지 설명합니다.
설정을 이해하면 더 정확한 논문을 더 효율적으로 수집할 수 있습니다.

키워드 처리 방식

키워드마다 별도 검색 후 결과를 병합합니다

키워드 여러 개를 입력하면 AND 조합이 아니라 각각 독립적으로 RISS를 검색합니다.

  • 예: "교사 정체성" + "교직 경험" → RISS를 두 번 검색 후 결과를 합칩니다
  • 같은 논문이 두 키워드에 모두 걸리면 먼저 검색된 키워드의 결과로 유지됩니다
  • 중복 판별 기준은 RISS 논문 고유번호(controlNo)입니다
  • 수집 결과 파일에는 각 논문이 어떤 키워드로 찾아졌는지 sourceKeyword 태그가 기록됩니다
팁: 비슷한 키워드를 여러 개 넣으면 더 넓게 수집할 수 있습니다. "교사 전문성"과 "교원 역량"을 함께 입력하면 각각 다른 표현으로 작성된 논문을 모두 포함합니다.
수집 파이프라인 전체 흐름

수집 → 병합 → 필터 → 분류 → 저장

아래 순서로 처리됩니다. 점선 박스는 설정에 따라 건너뛸 수 있는 단계입니다.

단계내용비고
① 키워드 검색 키워드마다 RISS 검색 페이지 수집, 페이지 내 중복 제거 키워드 수 × 페이지 수만큼 실행
② 결과 병합 모든 키워드 결과를 하나로 합치고 키워드 간 중복 제거 controlNo 기준
③ 필터 적용 제외 키워드·피인용 횟수·상위 N건 순서로 필터링 수집 완료 후 처리
④ AI 관련도 필터 Claude가 제목·초록을 읽고 관련 없는 논문 제외 선택 — Claude CLI/API 필요
⑤ PDF 다운로드 RISS → 외부 제공자(KCI·DBpia 등) 경유 다운로드 선택 — 도서관 로그인 필요
⑥ 주제 분류 전체 제목 분석 → 카테고리 자동 생성 → 폴더 분류 선택 — Claude CLI/API 필요
필터 적용 시점

일부 필터는 RISS 검색 시, 나머지는 수집 완료 후 적용됩니다

RISS 검색 URL에 직접 반영되는 필터 — 검색 결과 자체를 좁혀줍니다

설정RISS URL 파라미터
시작·종료 연도p_year1, p_year2
KCI 등재 논문만regnm=KCI등재
정렬 방식strSort=RANK / DATE / VIEWCOUNT

수집 완료 후 적용되는 필터 — 이미 수집된 결과를 걸러냅니다

설정동작 방식
제외 키워드제목 + 초록 텍스트에 해당 단어가 포함된 논문 제거 (대소문자 무시)
최소 피인용 횟수KCI 피인용수가 설정값 미만인 논문 제거
상위 N건피인용수 내림차순 정렬 후 N개만 추출
제외 키워드 활용 예시: "교사" 키워드로 검색하면 "유아교사"나 "대학생" 관련 논문도 포함될 수 있습니다. 제외 키워드에 "유아", "대학생"을 입력하면 수집 후 해당 단어가 제목이나 초록에 있는 논문이 제거됩니다.
AI 관련도 필터 (선택)

Claude가 논문과 키워드의 관련성을 직접 판단합니다

  • 각 논문의 제목과 초록을 읽고 입력한 키워드와의 관련도를 판정합니다
  • "관련 없음"으로 판정된 논문은 결과 목록에서 제외됩니다
  • 연구 맥락(B 섹션)을 입력했다면 그 내용도 판단 기준에 함께 활용됩니다
  • 15편씩 묶어 배치로 처리하므로 논문이 많을수록 시간이 걸립니다
  • 사용하려면 분류 건너뜀 체크를 해제하고 Claude CLI 또는 API 키가 설정되어 있어야 합니다
주제 분류 (선택)

수집된 논문을 주제별 폴더로 자동 분류합니다

  • 전체 논문 제목을 분석해 3~5개 카테고리를 자동으로 생성합니다
  • 각 논문을 카테고리에 배정한 후 해당 이름의 폴더로 PDF를 이동합니다
  • 카테고리명은 연구 내용을 반영해 Claude가 직접 작성합니다
예시: "교사 정체성" 키워드로 100편 수집 → 교직 입문과 전환 / 전문성 개발 / 문화자본과 배경 / 교육과정 실천 폴더 자동 생성 후 PDF 배분

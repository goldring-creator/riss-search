【 시작 전 확인 — Claude CLI 설치 여부 】
──────────────────────────────────────────

  논문 자동 분류 기능을 사용하려면 Claude CLI가 필요합니다.
  (분류 기능을 사용하지 않을 경우 건너뛰어도 됩니다.)


  ▶ 터미널(macOS) / 명령 프롬프트(Windows) 여는 방법

    [macOS]
      키보드에서 Command(⌘) + Space 를 누릅니다.
      "터미널" 이라고 입력하고 Enter 를 누르세요.
      검은색(또는 흰색) 창이 열리면 터미널입니다.

    [Windows]
      키보드에서 Windows 키 + R 을 누릅니다.
      "cmd" 라고 입력하고 Enter 를 누르세요.
      또는 시작 메뉴에서 "명령 프롬프트" 를 검색해도 됩니다.


  ▶ Claude CLI 설치 확인
     터미널(또는 명령 프롬프트)에 아래 명령어를 입력하고 Enter:

       claude --version

     결과가 어떻게 나오나요?

     ✅ 버전 번호가 표시된다면 (예: 1.x.x)
        → Claude CLI가 설치되어 있습니다. 아래 [사전 준비]로 이동하세요.

     ❌ "command not found" 또는 아무 반응이 없다면
        → 아래 설치 방법을 따라 주세요.


  ▶ Claude CLI 설치 방법

     1단계. Node.js 설치 확인
        아래 명령어 입력:

          node --version

        버전이 나오면 이미 설치된 것입니다. 2단계로 이동하세요.
        "command not found" 또는 오류가 뜨면 아래 주소에서 먼저 설치:
        https://nodejs.org  (LTS 버전 선택 → 다운로드 → 설치)
        설치 후 터미널을 닫았다가 다시 여세요.

     2단계. Claude CLI 설치
        아래 명령어 입력:

          npm install -g @anthropic-ai/claude-code

        설치가 완료될 때까지 기다립니다. (1~3분 소요)

     3단계. 설치 확인
        아래 명령어 입력:

          claude --version

        버전 번호가 표시되면 설치 성공입니다.

     4단계. 로그인
        아래 명령어 입력:

          claude

        처음 실행하면 Anthropic 계정 로그인 화면이 뜹니다.
        안내에 따라 로그인하세요.

     5단계. Claude CLI 접속 확인
        로그인 완료 후 터미널에 다시 아래 명령어를 입력하여
        Claude CLI에 정상 접속되는지 확인하세요:

          claude

        Claude 응답이 나오면 설치 및 로그인이 완료된 것입니다.
        터미널을 닫고 아래 [사전 준비]로 이동하세요.


================================================================

[ 사전 준비 ]

  1. Node.js 설치 (아직 없다면)
     https://nodejs.org → LTS 버전 다운로드 및 설치

  2. 이 폴더 위치 확인
     실행 파일과 "_app" 폴더가 같은 위치에 있어야 합니다.
     - macOS: "RISS 논문수집기.app" 또는 "RISS-search.command"
     - Windows: "RISS-search.bat"


================================================================

[ macOS — 첫 실행 방법 ]
  (macOS 보안 정책 때문에 최초 1회만 필요)

  방법 1) RISS 논문수집기.app
     → 아이콘을 우클릭(두 손가락 클릭) → "열기" 선택 → 팝업에서 "열기" 클릭
     이후부터는 더블클릭으로 바로 실행됩니다.

  방법 2) RISS-search.command (터미널 방식)
     → 파일을 우클릭 → "열기" 선택
     터미널 창이 열리고 자동으로 브라우저가 실행됩니다.


[ Windows — 첫 실행 방법 ]

  1. "RISS-search.bat" 파일을 더블클릭하세요.
     처음 실행 시 패키지 및 브라우저를 자동으로 설치합니다.
     (설치 시간: 2~5분, 최초 1회만 필요)

  2. 설치가 완료되면 브라우저가 자동으로 열립니다.

  ※ Windows Defender 또는 보안 경고가 뜨면:
     "추가 정보" → "실행" 을 클릭하면 됩니다.

  ※ 비밀번호는 Windows DPAPI(사용자 계정 암호화)로 저장됩니다.


================================================================

[ 사용 순서 — macOS / Windows 공통 ]

  1. 도서관 로그인
     - 한국외국어대학교 도서관 이용자 ID / PW 입력
     - "로그인 확인" 클릭 → 10~20초 후 자동으로 다음 화면으로 이동

  2. 검색 키워드 입력
     - 키워드 입력 후 Enter 또는 쉼표(,)로 태그 추가
     - 여러 키워드를 추가하면 각각 검색 후 결과를 합칩니다

  3. 필터 설정 (선택)
     - 수집 페이지 수, 연도 범위, KCI 등재 여부, 정렬 방식 등
     - Claude CLI 또는 API 키가 있으면 논문 자동 분류 기능 사용 가능

  4. 저장 위치 선택
     - "폴더 선택" 버튼으로 결과물 저장 위치 지정

  5. "수집 시작" 클릭
     - 수집 진행 상황이 로그 창에 실시간으로 표시됩니다
     - 완료 후 지정 폴더에 metadata.json / CSV / PDF 저장


[ 수집 결과물 ]

  - metadata.json  : 전체 논문 메타데이터
  - papers.csv     : Excel에서 바로 열 수 있는 목록
  - pdfs/          : 원문 PDF 파일 (파일명: APA 스타일)
                     예) 홍길동 외(2023). 제목. 학술지명, 권(호), 페이지.pdf

[ 자주 묻는 질문 ]

  Q. (macOS) "개발자를 확인할 수 없음" 오류가 뜨면?
  A. 아이콘을 우클릭 → "열기"를 선택하세요. 이후엔 더블클릭으로 실행됩니다.

  Q. (Windows) "Windows의 PC 보호" 경고가 뜨면?
  A. "추가 정보" 클릭 → "실행" 클릭하면 됩니다.

  Q. 앱이 안 열리거나 서버 오류가 뜨면?

    [macOS] 터미널에서:
      rm ~/.riss/port

    [Windows] 명령 프롬프트에서:
      del %USERPROFILE%\.riss\port

    그 다음 앱을 다시 실행하면 됩니다.

  Q. 비밀번호를 변경하고 싶으면?

    [macOS] 터미널에서:
      rm ~/.riss/config.json

    [Windows] 명령 프롬프트에서:
      del %USERPROFILE%\.riss\config.json

    앱 재실행 후 새 계정 정보 입력.

  Q. PDF가 일부 다운로드 안 되면?
  A. 해당 논문이 구독 범위 밖이거나 제공사 정책으로 제한된 경우입니다.
     metadata.json의 detailUrl을 브라우저에서 직접 열어 확인하세요.
